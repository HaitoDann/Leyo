# Leyo Shell

Leyo est un shell moderne pour Windows, conçu pour ceux qui aiment la puissance des commandes Linux mais qui travaillent dans l'écosystème Windows.

✨ Fonctionnalités Clés
Syntaxe Cross-Platform : Utilisez des commandes de type Unix (ls, rm, mkdir, cp) nativement sous Windows.

Dual-Pane View : Une interface divisée avec le prompt à droite et l'explorateur de fichiers dynamique à gauche.

Correction Intelligente : Si une commande échoue, Leyo analyse votre saisie et propose la commande la plus proche.

Autocomplétion Ghost : Suggestion de commandes en texte grisé basée sur votre historique, validable d'une simple pression sur la flèche droite.


🎨 Design & UI
Le shell utilise une palette de couleurs dégradées (Bleu/Violet) basée sur l'identité visuelle de l'hippocampe polygone.

Prompt : [utilisateur] C:\Chemin\Vers\Projet >

Sidebar : Liste interactive du contenu du dossier courant avec icônes de fichiers.

Feedback : Alertes d'erreurs en rouge avec suggestions en mode interactif (y/n).

Couleur syntaxique riche: Rouge (#FF4B5C) Erreurs; Vert (#2ECC71) Succès; Jaune (#F1C40F) Avertissement. Couleurs différentes dans la sidebar en fonction du format de fichier ou dossier.
