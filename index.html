Pourquoi Leyo ?

Tout ce qui manquait à cmd.exe et PowerShell

⌨️

Syntaxe Unix

Tape ls, rm, mkdir, cp — Leyo traduit automatiquement vers Windows.

💡

Correction intelligente

Algorithme de Levenshtein. Leyo détecte tes fautes de frappe et propose la bonne commande.

✨

Ghost text

Autocomplétion en texte grisé basée sur ton historique. Valide avec → ou Tab.

📁

Dual-pane

Sidebar fichiers interactive mise à jour en temps réel. Glisse-dépose vers le prompt.

🗂️

Onglets

Plusieurs sessions simultanées. Ctrl+T pour ouvrir, Ctrl+W pour fermer.

⎇

Git intégré

La branche Git courante s'affiche dans le prompt. Toujours au courant de ton contexte.

🎨

Interface native

Vraie fenêtre Windows avec effet Acrylic, palette low-poly inspirée du logo, prompt Powerline.

⚡

Alias persistants

Crée tes propres raccourcis sauvegardés entre les sessions. alias ll="ls -la"

🔍

Recherche Ctrl+F

Cherche dans tout l'output avec surlignage et navigation entre les occurrences.

👁️

Prévisualisation

Clic sur un fichier dans la sidebar pour le prévisualiser. Coloration par langage.

🎨

Thèmes

5 palettes incluses : PRISM, OCEAN, FOREST, SUNSET, MONO. Ctrl+K pour changer.

Installation

Deux minutes pour avoir Leyo sur ton Windows

Recommandé
📦

Installeur

Lance LeyoSetup.exe — Leyo s'installe avec un raccourci Bureau et une entrée dans le Menu Démarrer.

Télécharger LeyoSetup.exe Windows 10 / 11 — 64 bit
🗂️

Portable

Télécharge Leyo.exe, place-le dans C:\Leyo\ et lance-le directement.

Télécharger Leyo.exe Windows 10 / 11 — 64 bit

⚠️ Windows peut afficher un avertissement de sécurité — clique sur "Informations complémentaires" → "Exécuter quand même"
